# Task-0001: Configure Role-Based MCP Tool Filtering

**Status:** DONE
**Priority:** HIGH
**Assigned to:** Worker

## Description
The current tool filtering via environment variables (`JCODEMUNCH_DISABLED_TOOLS`) in `mcp_config.json` is being ignored by `jcodemunch-mcp` because of the existence of a global `~/.code-index/config.jsonc` file. This task involves moving these configurations into role-specific `config.jsonc` files and updating the initialization scripts to ensure they are correctly linked in both host and VM environments.
